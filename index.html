GitHub Agent MCP v2.0.0 - PowerShell Enhanced

Agente PowerShell inteligente com Model Context Protocol (MCP) para automação GitHub empresarial com cache, logs estruturados e métricas em tempo real

GitHub Agent CLI
⚡

Comandos PowerShell

Comandos nativos do PowerShell: gco, gcd, gcstats, gchelp com integração MCP completa

🧠

Cache Inteligente

Sistema de cache com TTL configurável, persistência entre sessões e validação automática de Node.js

📋

Logs Estruturados

Logs JSON com níveis (INFO, WARN, ERROR, DEBUG, METRIC), timestamps e session tracking

📊

Métricas MCP

Monitoramento em tempo real, performance tracking e error handling robusto com recovery automático

Comandos PowerShell MCP v2.0.0

. "scripts/load-agent-in-profile.ps1" Carregar agente PowerShell MCP
gco "feat: nova funcionalidade" Contribuição rápida com commit
gchelp Sistema de ajuda contextual
$Global:GitHubAgentConfig Configuração e status do agente
gcstats 60 Estatísticas dos últimos 60 dias
gcd Contribuição diária automatizada

Especificações MCP v2.0.0

Cache TTL: Node.js (60min), Configurações (30min)
Logs: JSON estruturado em $env:TEMP\GitHubAgent\logs
Compatibilidade: PowerShell 5.1+, Windows 10+, Node.js 18+
Validação: Automática de ambiente e dependências
